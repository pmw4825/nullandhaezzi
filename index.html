[믹스 공지사항]

1. '큐베이스'로 작업합니다. 파일 형식 [Wav 44,100Hz 24bit mono] 로 진행합니다.

2. 작업 순서 : 상담 → 입금 → 1차 완성 → 피드백(최대 3번) → 수정 → 최종 완성

3. 녹음본과 MR을 꼭 함께 보내주세요.

4. 폰 녹음도 받지만 음질 상태가 좋지 못한 건 감안해주세요.

5. 참고할 음원 링크를 보내주셔야 만족스러운 작업이 가능해요.
소통을 중요시 생각합니다. 원하는 느낌을 많이 말씀해주세요.

[영상 공지사항]

1. 작업은 (신청 → 중간 컨펌 → 최종 컨펌) 으로 진행되며, 커미션 주의 실수를 제외한 이 외의 수정은 불가합니다.

2. 'After effect 2023'으로 작업하며 작업물은 1920 x 1080, 24 or 30fps, mp4 형식으로 제공됩니다.

3. 전신 일러스트가 아니거나, 일부가 잘린 경우 연출이 다소 제한될 수 있습니다.

4. 커미션 주의 영상 스타일 특성상 어둡고, 빠른 곡이 가장 잘 어울립니다.

5. 레퍼런스 영상이나 원하시는 분위기를 자세히 말씀해주시면 제작에 도움됩니다.
